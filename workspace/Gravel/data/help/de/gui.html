Die grafische Oberfläche


Abbildung 1: Das Hauptfenster zu Beginn

Die grafische Oberfläche (engl.: graphical user interface, kurz GUI) stellt die Schnittstelle zum Benutzer dar. Zunächst wird beim Start das Hauptfenster initialisiert. Beim ersten Start bzw. wenn zu Beginn kein Graph geladen wird, erhalten wir ein Hauptfenster, wie in Abbildung 1. Unter Windows bzw. Linux ist dabei noch das Menü in dem Fenster am oberen Rand. Unter Mac OS X wird das Menü am oberen Bildschirmrand verwendet.
Die Einträge des Menüs sind auch über Tastenkürzel erreichbar. Daher ist das Menü dort erläutert.

Die Darstellung des Graphen im Hauptfenster stellt für jedes Element ein Kontextmenü zur Verfügung sowie verschiedene Dialoge.
